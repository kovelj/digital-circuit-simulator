# Digital Circuit Simulator

### Time domain digital circuit simulation engine

Simulation of digital circuit output in any arbitrarily position for probe. Circuit is loaded from .txt file 
which contains information about connection between digital components, position of probe and input signal in 
time domain.

Engine loades circuit, connect components and after that for probe position output is simulated.

For this repo some pictures would be nice.
Project is written in **Visual Studio 2019**.
